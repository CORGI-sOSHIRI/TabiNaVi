## 🐾 たびNaViとは？

旅行のしおりを **コーギーと簡単に** 作れるアプリです 
複数人で予定を共有しながら、旅の思い出を形に残せます。

### 主な機能

- 1日のタイムスケジュールを可視化  
- 目的地を自動でマップにピン留め  
- 複数人で編集  
- 想い出のしおりを保存
